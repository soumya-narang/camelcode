Welcome to CamelCode! In this first lesson, you'll write your first OCaml program and learn about the OCaml toplevel (also known as the REPL - Read-Eval-Print Loop).

## What is OCaml?

OCaml is a general-purpose, high-level programming language with an emphasis on expressiveness and safety. It's a functional programming language, which means it treats computation as the evaluation of mathematical functions.

## Your First Program

Let's start with the classic "Hello World" program. In OCaml, we use `print_endline` to output text:

```ocaml
print_endline "Hello, World!";;
```

Try it yourself! Click the **Run Code** button below to see the output.

## Understanding the Code

Let's break down what happened:

- `print_endline` is a function that prints a string followed by a newline
- `"Hello, World!"` is a string literal
- `;;` marks the end of an expression (optional in files, but common in the toplevel)

## Try It Yourself

Modify the code below to print your own message:

```ocaml
print_endline "Your message here";;
```

## Key Takeaways

- OCaml uses `print_endline` for console output
- Strings are enclosed in double quotes
- The toplevel evaluates expressions and shows their results
- You can experiment with code interactively
