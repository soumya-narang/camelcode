# camelcode

An interactive learning platform for OCaml — structured modules from beginner to intermediate, with a built-in editor to practice as you learn.

## The problem

OCaml has no single place that teaches it properly. The official docs assume you already know functional programming. Everything else is scattered, outdated, or incomplete.

## What camelcode is

- **Structured modules** — concepts introduced progressively, sourced strictly from official and verified resources
- **Inline editor** — practice each concept without leaving the page
- **No fluff** — straight to understanding, then straight to writing code

## Status

In development. Contributions welcome.
